◆ exclude

bent_comp_reg_t** bent_sys_def_t::exclude

Null-terminated list of required components.

Defaults to an empty list which will match all entities.

An entity must have none of the listed components to be processed by this system.

See also
BENT_COMP_LIST
Remarks
If both this and bent_sys_def_t::require are NULL, this system will not match any entities.

◆ post_init

void(* bent_sys_def_t::post_init) (void *userdata, bent_world_t *world)

Optional post-initialization callback.

Unlike bent_sys_def_t::init, this will always be called after every system has finished initialization. This allows a dependent system to make queries to another system. The bent_sys_def_t::allow_reinit flag has no effect.

bent_comp_reg_t** bent_sys_def_t::require

Null-terminated list of required components.

Defaults to an empty list which will match all entities.

An entity must have all of the listed components to be processed by this system.

◆ size

size_t bent_sys_def_t::size

Size of the data for this system.

If not specified it will be treated as 0. In which case, no extra storage is allocated.

Remarks
This property is not hot reloadable.
