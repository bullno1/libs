◆ cleanup

void(* bent_comp_def_t::cleanup) (void *comp)

Optional initialization callback for this component.

If this is omitted, nothing will be done upon destruction. Take note that the component's storage memory is owned by the library and it can be recycled.

◆ init

void(* bent_comp_def_t::init) (void *comp, void *arg)

Optional initialization callback for this component.

If this is omitted, the default behavior is as follow:

  • If arg is NULL, the component's data will be zero-ed.
  • If arg is not NULL, arg will be memcpy-ed into the component.

◆ size

size_t bent_comp_def_t::size

Size of the data for this component.

If not specified it will be treated as 0. In which case, no extra storage is allocated. This component can be used purely for "tagging".
