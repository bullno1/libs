Detailed Description

A single header library for resource monitoring and reloading.

In exactly one source file, define BRESMON_IMPLEMENTATION before including bresmon.h.

Optionally, define BRESMON_REALLOC(ptr, size, ctx) to override the allocator. By default, libc will be used.
