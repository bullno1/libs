Detailed Description

Type-safe hashtable.

Based on: https://nullprogram.com/blog/2022/08/08/

Several functions in this library return an index. If it is -1, it indicates a "not found" result. Otherwise, it is an index into the bhash_sample_t::keys and ref bhash_sample_t.values arrays.
