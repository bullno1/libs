Detailed Description

Crash handler.

Set a function to be called before the program crashes. bstacktrace.h can be used to generate a stacktrace of the crash.

Supported platforms:

  • Windows
  • Linux

BCRASH_HANDLER_API bool bcrash_handler_should_report_current_frame ( bcrash_info_t *  info)

Check whether the current stackframe inside a stack walker should be reported.

The crash handler itself also has a call stack which is not relevant for debugging. Different platforms have different stack depth for crash handling. This function helps to smooth over the differences.

This function is designed to be called inside a bstacktrace's callback

Example:

static bool
stack_walk(uintptr_t address, void* userdata) {
bcrash_info_t* crash_info = userdata;
// Skip frames belonging to the crash handler system
if (!bcrash_handler_should_report_current_frame(crash_info)) { return true; }

BCRASH_HANDLER_API bool bcrash_handler_should_report_current_pc ( bcrash_info_t *  info)

Check whether the code location at bcrash_info_t::pc should be reported.

Differentt platforms invoke the crash handler differently. The stacktrace originating from the crash handler may or may not encompass the crash site. This function helps to smooth over the differences.
